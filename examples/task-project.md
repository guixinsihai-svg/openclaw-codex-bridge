# Project task example

## Goal

Make one small, reviewed documentation change inside the project workspace passed to `scripts/run_project_task.ps1`.

## Scope

Allowed:

- Read files inside the provided workspace.
- Edit documentation files inside the provided workspace only.

Not allowed:

- Do not edit files outside the provided workspace.
- Do not delete files.
- Do not modify credentials, `.env`, tokens, API keys, scheduled tasks, databases, OpenClaw, or n8n.
- Do not publish, upload, or push anything.

## Expected result

Summarize:

- which files were inspected,
- which files were changed,
- what remains for human review.
